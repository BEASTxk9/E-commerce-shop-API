API ENDPOINTS:


PRODUCTS ENDPOINTS


  • (GET)/products :Get all products.
  • (GET)/products/:id : Get single product.
  • (POST)/products/:id : Add single product.
  • (PUT)/products/:id : Update single product.
  • (DELETE)/products/:id : DELETE single product.

    • USERS ENDPOINTS


  • (GET)/users :Get all users.
  • (GET)/users/:id : Get single user.
  • (POST)/users/:id : Add single user.
  • (PUT)/users/:id : Update single user.
  • (DELETE)/users/:id : DELETE single user.

    • CART ENDPOINTS


  • (GET)/users/:id/cart :Get all user cart items.
  • (GET)/users/:id/cart/:cart_id : Get single user cart item.
  • (POST)/users/:id/cart : Add single item to cart.
  • (DELETE)/users/:id/cart : DELETE all items from user cart.
  • (DELETE)/users/:id/cart/:cart_id : DELETE single item from user cart.
